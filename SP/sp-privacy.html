プライバシーポリシー

Dorado合同会社（事業所名：More Friends）は、利用者の皆様からお預かりする個人情報の重要性を認識し、適正な取り扱いを行います。

制定日：2025年12月1日

第1条（基本方針）

Dorado合同会社（事業所名：More Friends）（以下「当社」といいます。）は、当社が提供する訪問型ペットシッティングサービス（以下「本サービス」といいます。）の運営にあたり、利用者の皆様からお預かりする個人情報の重要性を認識し、個人情報の保護に関する法律（以下「個人情報保護法」といいます。）その他の関係法令およびガイドラインを遵守し、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定め、個人情報を適正に取り扱います。

第2条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報保護法第2条第1項に定める個人情報をいい、生存する個人に関する情報であって、氏名、住所、電話番号、メールアドレスその他の記述等により特定の個人を識別できるもの、または個人識別符号が含まれるものをいいます。

第3条（取得する情報）

当社は、本サービスの提供にあたり、以下の情報を取得することがあります。

利用者に関する情報

  1. 氏名、住所、電話番号、メールアドレスその他の連絡先情報
  2. 本人確認のために提示または提出いただく書類に記載された情報
  3. 緊急連絡先として届出いただいた方の氏名および連絡先
  4. サービス提供場所（利用者宅等）の所在地および住宅環境に関する情報
  5. 合鍵およびキーボックスに関する情報（預託方法、設置場所、暗証番号等）
  6. 決済に必要な情報（銀行口座情報等。ただし、クレジットカード情報については決済サービス提供元であるSquare, Inc.が管理しており、当社は保持しません。）

ペットに関する情報

  1. ペットの種類、頭数、名前、年齢、性別
  2. 性格、行動特性、飼育状況
  3. 健康状態、既往症、現在の病気・怪我・体調不良の有無
  4. 感染症の有無および寄生虫症に関する情報
  5. ワクチン接種状況およびその証明書類
  6. 投薬に関する情報（薬剤の種類、投薬量、投薬時刻、投薬手順等）
  7. かかりつけ動物病院の名称および連絡先

サービス提供に伴い取得する情報

  1. 事前カウンセリングにおけるヒアリング内容および記録
  2. サービス提供中に撮影したペットの写真
  3. サービス実施状況に関するレポートおよび業務記録
  4. メッセージングアプリ（LINE等）、電子メール、電話等を通じたやり取りの記録
  5. Instagram等のSNSを通じたお問い合わせ・やり取りの記録

ウェブサイト等を通じて取得する情報（当社ウェブサイト開設後に適用）

  1. Cookie（クッキー）およびこれに類する技術により取得される閲覧履歴、アクセスログ等の情報
  2. アクセス解析ツール（Googleアナリティクス等）により収集されるトラフィックデータ
  3. 広告配信サービスを通じて取得されるアクセス情報

第4条（情報の取得方法）

当社は、以下の方法により個人情報を取得します。

  • メッセージングアプリ（LINE等）、電子メール、電話、Instagram等のSNS、その他当社が認める手段を通じた申込み・問い合わせ
  • 事前カウンセリング（対面によるヒアリング）
  • 申込書面、見積書、サービス内容説明書その他当社が提示する書面への記入
  • 本人確認書類の提示または提出
  • サービス提供中の現場での確認・記録
  • 決済サービス（Square等）を通じた決済処理に伴う取得
  • 当社ウェブサイトにおけるCookie等の自動取得技術（ウェブサイト開設後に適用）

第5条（利用目的）

当社は、取得した個人情報を、以下の目的の範囲内で利用します。

  • 本サービスの申込受付、事前カウンセリングの実施、契約の締結および履行
  • 本人確認の実施
  • サービス提供日時の調整、訪問スケジュールの管理
  • サービス提供中および提供後の利用者への連絡・報告（レポート・写真の送付を含む）
  • 緊急時における利用者、緊急連絡先、動物病院、警察、消防その他関係機関への連絡・対応
  • 投薬その他利用者の指示に基づく対応
  • 合鍵の預託、管理、返却に関する対応
  • 利用料金の算定、請求、決済処理、返金対応
  • 消耗品の不足時における買い物代行および費用精算
  • サービスの品質向上、業務改善、トラブル防止
  • 事例紹介等を目的としたウェブサイト、SNS、ブログ、資料等への写真・レポートの掲載（第8条に定める条件に従います。）
  • 当社スタッフ、業務委託先および代行スタッフへの業務遂行に必要な範囲での情報共有
  • 保険会社への保険金請求に関する対応
  • 紛争解決、法的手続への対応
  • 法令に基づく対応
  • 当社ウェブサイトの運営・改善、アクセス状況の分析、広告の配信・効果測定（ウェブサイト開設後に適用）

第6条（第三者提供）

当社は、以下のいずれかに該当する場合を除き、あらかじめ利用者の同意を得ることなく、個人情報を第三者に提供しません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、利用者の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、利用者の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、利用者の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

第7条（業務委託・代行・緊急時の情報共有）

当社は、本サービスの全部または一部を第三者に委託する場合、または代行スタッフにより対応する場合があります。この場合、業務遂行に必要な範囲で、以下の情報を委託先または代行スタッフに提供することがあります。

  1. 利用者の氏名、住所、連絡先
  2. ペットおよび利用者に関するカウンセリング情報（健康状態、性格、注意事項等）
  3. サービス内容および実施条件
  4. 合鍵またはキーボックスに関する情報

当社は、前項の情報を提供するにあたり、委託先または代行スタッフとの間で守秘義務および情報管理義務を課す契約を締結し、適切な監督を行います。

緊急時対応として、以下の場合に、利用者の個人情報またはペットに関する情報を関係先に提供することがあります。

  1. ペットの体調不良、怪我、事故等が生じた場合における、かかりつけ動物病院または近隣の動物病院への情報提供
  2. 利用者と連絡が取れない場合における、緊急連絡先への連絡
  3. 火災、ガス漏れ、侵入被害その他の異常が生じた場合における、警察、消防その他関係機関への通報・情報提供
  4. 保険金請求に関連して、当社が加入する賠償責任保険の保険会社への情報提供

クレジットカード決済に関しては、決済サービス提供元であるSquare, Inc.に対し、決済処理に必要な範囲で情報が提供されます。当社はクレジットカード番号等の決済情報を保持しません。

第8条（写真・レポート・記録の取扱い）

当社は、本サービスの提供および品質向上、業務記録、トラブル防止を目的として、ペットおよびサービス提供状況に関する写真または簡易な記録を取得することがあります。

前項の写真およびレポートは、利用者への報告・共有のために送付することがあります。撮影は、ペットおよびサービスに関連する範囲に限定し、利用者のプライバシーに配慮して行います。

当社は、取得した写真およびレポートを、当社のサービス品質向上、業務改善、事例紹介等を目的として利用することがあります。当該利用には、当社が運営するウェブサイト、ブログ、SNS、資料等への掲載が含まれる場合があります。

利用者は、写真およびレポートの公開利用を希望しない場合には、あらかじめ当社に申し出ることにより、当該利用を拒否することができます。当社は、当該申出があった場合、公開目的での利用を行いません。

第9条（安全管理措置）

当社は、取得した個人情報の漏えい、滅失、毀損等を防止するため、以下の安全管理措置を講じます。

  • 組織的措置：個人情報保護の責任者を設置し、個人情報の取扱いに関する管理体制を整備します。
  • 人的措置：当社スタッフに対し、個人情報の適正な取扱いに関する教育を実施し、守秘義務を課します。業務委託先および代行スタッフに対しても、守秘義務および情報管理義務に関する契約を締結します。
  • 物理的措置：合鍵は専用キーボックスを通じて管理し、個人情報を記載した書類は施錠可能な場所に保管する等、紛失・盗難防止のための措置を講じます。
  • 技術的措置：個人情報を取り扱う端末へのアクセス制限、パスワード管理その他不正アクセス防止のための措置を講じます。

第10条（個人情報の保管期間）

当社は、利用目的の達成に必要な期間、個人情報を保管します。利用目的を達成し、保管の必要がなくなった場合は、合理的な期間内に適切な方法により消去または廃棄します。ただし、法令に基づき保管が義務づけられている場合は、当該法令の定めに従います。

第11条（Cookie・アクセス解析・広告について）

※ 本条は、当社ウェブサイト開設後に適用されます。

Cookieについて
当社のウェブサイトでは、利用者の利便性向上およびウェブサイトの改善のため、Cookie（クッキー）を使用することがあります。Cookieとは、ウェブサイトへのアクセスに関する情報をブラウザに保存する仕組みであり、氏名、住所、電話番号等の個人情報は含まれません。利用者は、ブラウザの設定によりCookieの受け入れを拒否することができますが、一部の機能が利用できなくなる場合があります。

アクセス解析ツールについて
当社のウェブサイトでは、Google LLCが提供するアクセス解析ツール「Googleアナリティクス」を利用することがあります。Googleアナリティクスは、Cookieを使用してトラフィックデータを収集しますが、当該データは匿名で収集されており、特定の個人を識別するものではありません。利用者は、ブラウザの設定またはGoogleアナリティクスオプトアウトアドオンの導入により、データの収集を拒否することができます。

広告配信について
当社のウェブサイトでは、Google LLC、Meta Platforms, Inc.その他の広告配信事業者のサービスを利用し、広告を配信することがあります。これらのサービスでは、Cookieや広告識別子等を利用して、利用者の過去のアクセス情報に基づく広告を配信する場合があります。利用者は、各広告配信事業者の設定画面からパーソナライズ広告を無効にすることができます。

第12条（開示・訂正・利用停止等）

利用者は、当社に対し、個人情報保護法の定めに基づき、自己に関する保有個人データについて、以下の請求を行うことができます。

  1. 利用目的の通知
  2. 開示
  3. 内容の訂正、追加または削除
  4. 利用の停止または消去
  5. 第三者への提供の停止

前項の請求を行う場合は、第14条に定める問い合わせ窓口までご連絡ください。当社は、請求者がご本人であることを確認のうえ、合理的な期間内に対応いたします。

利用の停止または消去を行った結果、本サービスの全部または一部を提供できなくなる場合がありますので、あらかじめご了承ください。

第13条（本ポリシーの変更）

当社は、必要と判断した場合には、本ポリシーの内容を変更することがあります。

本ポリシーの変更は、当社が利用者に対し、電子メール、メッセージングアプリ、当社ウェブサイトその他当社が適切と判断する方法により通知した時点から効力を生じるものとします。
